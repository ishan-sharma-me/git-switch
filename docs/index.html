git-switch

Switch between multiple GitHub identities in one command.

Heads up: This project was coded with AI assistance. Please be cautious, especially if you are an open source maintainer with critical repo access or commit signing responsibilities. A comprehensive security audit is pending. Review the source and test in a safe environment first.
curl -fsSL https://ishan-sharma-me.github.io/git-switch/install.sh | bash

macOS (Intel & Apple Silicon) and Linux (x64 & ARM64)

curl -fsSL https://ishan-sharma-me.github.io/git-switch/uninstall.sh | bash

SSH config

Manages ~/.ssh/config safely with markers. Your other hosts are never touched.

GPG signing

Per-account GPG keys. Signing is enabled or disabled automatically on switch.

Shell completions

Tab-complete your account names. Supports zsh, bash, and fish.

Claude Code

Run git-switch ai to teach Claude Code about your identities.
